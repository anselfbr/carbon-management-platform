Rule Master Fields

Rule Type Guide

Plant Exact Plant 完全符合，例如：3760 → 石碣廠-IPS
Material Number Exact 完整料號比對，例如：SG-96000-00A
Material Number Prefix 料號前綴比對，例如：850-、851-、852-
Series Exact 完整產品系列比對，例如：SN3103B02
Series Prefix 產品系列前綴比對，例如：SN、SP、FU、SCMC
Description Contains Material description 包含關鍵字，例如：TOUCHPAD MODULE、ASSY

Maintenance Principles

  • Priority 數字越小，優先度越高。
  • Product Line 與 Production Site 僅由 Rule Master 控制。
  • 新增或調整分類規則時，只需修改對應 BU 的 rule_master.csv，不需修改 main.py。
  • 若 Product Line 或 Production Site 空白，平台會保持空白，不自動推論。
